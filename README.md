## 📌 About Project

College Predictor is a software application developed to help students during the engineering admission and counseling process.  
The system takes the student's JEE Mains rank/marks as input and compares it with cutoff data to predict possible engineering colleges and branches the student may get.

The project provides a simple and user-friendly interface, making it easier for students to explore admission possibilities without manually checking cutoff lists.  
It is designed to save time and guide students in making better decisions during college selection.
